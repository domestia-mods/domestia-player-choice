# Domestia Player Choice

Fabric scaffold for a player-facing digest and notice board system.

Current state: project skeleton only. Dynamic catalog parsing, scrollable GUI, image page rendering, and two-block board behavior are intentionally left as TODO stubs.

Content boundary:

- `domestia_player_choice.jar` provides items, blocks, GUI logic, parser, and bundled fallback content.
- `domestia-player-choice-content.zip` provides the live `catalog.json` and page images through a server or local resource pack.
